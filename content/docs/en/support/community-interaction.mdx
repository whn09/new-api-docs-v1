Welcome to the New API open-source community! Here, you can exchange experiences and share ideas with other developers.

## üìú Group Rules

To maintain a positive community environment, please observe the following rules:

<Callout type="warn" title="Important Notice">
  Violation of these rules may result in removal from the chat group. All members are requested to read and comply carefully.
</Callout>

1.  Group administrators and the group owner are under no obligation to provide you with any technical support. If you have questions, please submit an [issue](feedback-issues.md).
2.  The group chat prohibits posting any information related to account sales or purchases. Group nicknames are prohibited from containing any purchase guidance or solicitation information.
3.  This group does not sell any API products. Please do not trust or purchase API products from anyone (including administrators).
4.  New API does not have a paid version.
5.  This project must be used in compliance with relevant laws and regulations. Discussion of any illegal topics is prohibited.

<Callout title="Friendly Reminder">
  If you have any feature suggestions or bug feedback, it is recommended to submit an issue directly on GitHub for better tracking and resolution.
</Callout>

            if (wrongAnswers.length > 0) {
                // Display detailed information for the first incorrect answer
                errorMessage.innerHTML = errorMessages[wrongAnswers[0]];
                quizError.style.display = 'block';

                // Turn button red and add shake animation
                submitBtn.style.background = '#c62828';
