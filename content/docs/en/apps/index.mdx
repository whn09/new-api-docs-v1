## Integration Guide

Most applications only need to configure the following information to connect to New API:

1. **API URL**: Your New API service address
2. **API Key**: Token generated in the New API console
3. **Model Name**: Select the model you want to use

Please refer to each application's detailed documentation for specific configuration steps.
