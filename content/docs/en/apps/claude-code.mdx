## AI Model Configuration Method

### Windows Graphical Guide

#### 1. Install Node.js Environment

Claude Code requires the Node.js environment to run.

<Callout type="info" title="Node.js Environment Installation Steps">
  - Open your browser and visit https://nodejs.org/ - Click to download the
  "LTS" version (Long-Term Support version recommended) - After the download is
  complete, double-click the .msi file - Follow the installation wizard to
  complete the installation, keeping the default settings
</Callout>

!!! caution "Windows Notes" - It is recommended to use PowerShell instead of CMD - If you encounter permission issues, try running as administrator - Some antivirus software may flag it incorrectly; you might need to add it to a whitelist

#### 2. Install Git Bash

!!! caution "Windows Notes"
In the Windows environment, Git Bash is required to install Claude Code. After installation, environment variable setup and using Claude Code should still be done in standard PowerShell or CMD.

{/* TODO: Convert to Callout - info: Download and Install Git for Windows */} - Visit https://git-scm.com/downloads/win - Click "Download for Windows" to download the installer - Run the downloaded .exe installation file - Keep the default settings during the installation process and click "Next" to complete the installation

    `
If the version number is displayed, the installation was successful

#### 3. Install Claude Code

<Callout type="info" title="Install Claude Code">
  Open PowerShell and run the following command:
</Callout>
``` npm install -g @anthropic-ai/claude-code ``` This command downloads and
installs the latest version of Claude Code from the official npm repository.

![windows-img-09.webp](/assets/claude_code/windows-img-09.webp)

{/* TODO: Convert to Callout - info: Add ~/.local/bin to PATH (Only if prompted) */}
`     [Environment]::SetEnvironmentVariable('Path', ([Environment]::GetEnvironmentVariable('Path','User') + ";$HOME\.local\bin"), 'User')
    `

{/* TODO: Convert to Callout - info: Verify Claude Code Installation */}
After installation is complete, enter the following command to check if the installation was successful:
`     claude --version
    `
If the version number is displayed, congratulations! Claude Code has been successfully installed.

#### 4. Set Environment Variables

<Callout type="info" title="One-Click Setup Command (Windows System)">
  To allow Claude Code to connect to your relay service, you need to set
  multiple environment variables:
</Callout>

```

进入你的项目目录 cd C:\path\to\your\project ``` ``` # 启动 Claude Code claude

Open Terminal

#### 6. macOS Common Issues Resolution

{/* TODO: Collapsible example: macOS Security Settings Block Execution */}
If the system prevents Claude Code from running:

    - Click "Open Anyway" or "Allow"
    - Or run in `Terminal`: `sudo spctl --master-disable`
