## Overview

New API is a next-generation large model gateway and AI asset management system, developed based on [One API](https://github.com/songquanpeng/one-api). This project provides a unified interface for managing and utilizing various AI model services.

<Callout type="info">
  New API seamlessly integrates with major AI platforms, including OpenAI,
  Anthropic, Google Gemini, Midjourney, and many other services.
</Callout>

### Core Features

- **Unified Interface** - A single API endpoint for multiple AI providers
- **Secure Management** - Advanced authentication and authorization
- **Usage Analytics** - Comprehensive tracking and reporting
- **High Performance** - Optimized for speed and reliability
- **Multi-Tenant Support** - Perfectly suited for teams and enterprises
