The system uses a four-level authentication mechanism: Public, User, Admin, Root
The AccessToken is the token generated by you **in Personal Settings - System Access Token** for accessing the **/api** route.

## 🔐 Authentication

- Public: No login required
- User: Requires User AccessToken (middleware.UserAuth)
- Admin: Requires Admin AccessToken (middleware.AdminAuth)
- Root: Limited to highest privilege user AccessToken (middleware.RootAuth)
