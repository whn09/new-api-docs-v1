## 概述

New API 提供完整的 RESTful API 接口，分为 **AI 模型接口** 和 **管理接口** 两大类。您可以通过这些接口实现 AI 能力调用和系统管理功能。

<Callout title="在线调试">
  您可以访问 [Apifox 操练场](https://apifox.newapi.ai/) 在线测试和调试 API 接口，或浏览下方的 API 文档。
</Callout>
