## 💡 我们的初衷

New API 诞生的初衷很简单：**让更多人能够使用 AI，降低 AI 的门槛**。

我们相信，AI 不应该只是少数人的特权，而应该成为每个人都能触及的工具。无论你是独立开发者、初创团队，还是大型企业，都应该能够以合理的成本接入全球领先的大模型能力。

## 🤝 关于商业化

我们选择商业化，并非为了追求利润最大化，而是为了让项目能够**良性、可持续地发展**。

服务器需要成本，维护需要精力，生态建设需要资源——适度的商业化是让 New API 持续为社区服务的必要保障。我们承诺将收益投入到产品迭代、社区建设和生态拓展中，让每一分支持都能转化为更好的服务。

## 📜 君子协议

New API 社区一直秉承**君子协议**的理念：

- 我们信任每一位用户，提供灵活开放的使用方式
- 我们不设置繁琐的限制，相信大家会合理使用资源
- 我们期待用户在受益的同时，也能尊重项目的可持续发展
- 如果您的项目因 New API 而获益，欢迎以任何形式回馈社区

这是一份基于信任与尊重的约定，也是开源精神的延续。
