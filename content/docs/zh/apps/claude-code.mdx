## AI 模型配置方法

### Windows 端图文指引

#### 1.安装 Node.js 环境

Claude Code 需要 Node.js 环境才能运行。

{/* TODO: Convert to Callout - info: Node.js 环境安装步骤 */} - 打开浏览器访问 https://nodejs.org/ - 点击 "LTS" 版本进行下载（推荐长期支持版本）- 下载完成后双击 .msi 文件 - 按照安装向导完成安装，保持默认设置即可

!!! caution "Windows 注意事项" - 建议使用 PowerShell 而不是 CMD - 如果遇到权限问题，尝试以管理员身份运行 - 某些杀毒软件可能会误报，需要添加白名单

!!! caution "验证安装是否成功"
安装完成后，打开 PowerShell 或 CMD，输入以下命令：
`     node --version
    npm --version
    `
如果显示版本号，说明安装成功

#### 2.安装 Git Bash

!!! caution "Windows 注意事项"
Windows 环境下需要使用 Git Bash 安装 Claude code。安装完成后，环境变量设置和使用 Claude Code 仍然在普通的 PowerShell 或 CMD 中进行。

{/* TODO: Convert to Callout - info: 下载并安装 Git for Windows */} - 访问 https://git-scm.com/downloads/win - 点击 "Download for Windows" 下载安装包 - 运行下载的 .exe 安装文件 - 在安装过程中保持默认设置，直接点击 "Next" 完成安装

![windows-img-06.webp](/assets/claude_code/windows-img-06.webp)

![windows-img-07.webp](/assets/claude_code/windows-img-07.webp)

![windows-img-08.webp](/assets/claude_code/windows-img-08.webp)

!!! caution "验证 Git Bash 安装"
安装完成后，打开 Git Bash，输入以下命令验证：
`     git --version
    `
如果显示版本号，说明安装成功

#### 3.安装 Claude Code

{/* TODO: Convert to Callout - info: 安装 Claude Code */}
打开 PowerShell，运行以下命令：
`     npm install -g @anthropic-ai/claude-code
    `
这个命令会从 npm 官方仓库下载并安装最新版本的 Claude Code。

![windows-img-09.webp](/assets/claude_code/windows-img-09.webp)

{/* TODO: Convert to Callout - info: 将 ~/.local/bin 加入 PATH（仅当提示要求时） */}
`     [Environment]::SetEnvironmentVariable('Path', ([Environment]::GetEnvironmentVariable('Path','User') + ";$HOME\.local\bin"), 'User')
    `

{/* TODO: Convert to Callout - info: 验证 Claude Code 安装 */}
安装完成后，输入以下命令检查是否安装成功：
`     claude --version
    `
如果显示版本号，恭喜你！Claude Code 已经成功安装了。

#### 4.设置环境变量

{/* TODO: Convert to Callout - info: 一键设置命令 (Windows 系统) */}
为了让 Claude Code 连接到你的中转服务，需要设置多个环境变量：

```powershell
iex (irm 'https://raw.githubusercontent.com/QuantumNous/new-api-docs/refs/heads/main/helper/claude-cli-setup.ps1')
```

![windows-configure](/assets/claude_code/windows_configure.png)

#### 5.开始使用 Claude Code

现在你可以开始使用 Claude Code 了！

{/* TODO: Convert to Callout - info: 启动 Claude Code */}
打开 PowerShell，直接启动 Claude Code:
`     # 启动 Claude Code
    claude
    `
在特定项目中使用:
`     # 进入你的项目目录
    cd C:\path\to\your\project
    `
`     # 启动 Claude Code
    claude
    `

#### 6.macOS 常见问题解决

{/* TODO: Collapsible example: macOS 安全设置阻止运行 */}
如果系统阻止运行 Claude Code：

    - 打开"系统偏好设置" → "安全性与隐私"
    - 点击"仍要打开"或"允许"
    - 或者在`Terminal`中运行：`sudo spctl --master-disable`
