在 New API 控制台的系统设置->聊天设置中，可添加如下快捷选项，便于在令牌管理页一键填充到 Cherry Studio：

    ```json
    { "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
    ```

### 参数填写

API 密钥：于 NewAPI 获取  
API 地址：NewAPI 站点地址
