## 統合ガイド

ほとんどのアプリケーションは、New API に接続するために以下の情報を設定するだけで済みます。

1.  **API アドレス**：お使いの New API サービスアドレス
2.  **API Key**：New API コンソールで生成されたトークン
3.  **モデル名**：使用するモデルを選択

具体的な設定手順については、各アプリケーションの詳細ドキュメントをご参照ください。
