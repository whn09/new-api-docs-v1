## 統合ガイド

ほとんどのアプリケーションは、以下の情報を設定するだけで New API に接続できます：

1. **APIアドレス**：New API サービスのアドレス
2. **APIキー**：New API コンソールで生成したトークン
3. **モデル名**：使用するモデルを選択

具体的な設定手順については、各アプリケーションの詳細ドキュメントを参照してください。
