<Callout type="info" title="チャット設定オプション">
  New API
  コンソールの「システム設定」->「チャット設定」で、以下のショートカットオプションを追加できます。これにより、トークン管理ページからFluentReadにワンクリックで入力できるようになります。
</Callout>

    ```json
    { "流畅阅读": "fluentread" }
    ```

<Callout type="info">
  🌊
  FluentReadは、母国語のような読書体験を提供することを目指したオープンソースのブラウザ翻訳プラグインです。
</Callout>

    - プロジェクトURL：`<https://github.com/Bistutu/FluentRead>`

## 🌟 主要機能

### スマート翻訳エンジン

- **マルチエンジン対応**：20種類以上の翻訳エンジンをサポート
- **従来の翻訳**：Microsoft翻訳、Google翻訳、DeepL翻訳など
- **AI大規模モデル**：OpenAI、DeepSeek、Kimi、Ollamaなど
- **カスタムエンジン**：カスタム翻訳サービスの設定をサポート

### 没入型読書体験

- **バイリンガル表示**：原文と翻訳文が並列表示され、より読みやすくなります
- **選択翻訳**：任意のテキストを選択すると、即座に翻訳結果が得られます
- **ワンクリックコピー**：翻訳文を素早くコピーし、読書効率を向上させます
- **全文翻訳**：フローティングボタンでウェブページ全体をワンクリックで翻訳でき、ページのリフレッシュは不要です

### プライバシーとカスタマイズ

- **プライバシー保護**：すべてのデータはローカルに保存され、コードはオープンソースで透明です
- **高度なカスタマイズ**：豊富なカスタムオプションで、さまざまなシナリオのニーズに対応します
- **完全無料**：オープンソースで無料で、非営利プロジェクトです

## 🚀 設定方法

### New API コンソールから設定をインポート（推奨）

ブラウザにFluentReadプラグインがインストールされている場合、New API コンソール->トークン管理ページを開くと、FluentReadを追加するプロンプトが表示されます。

![添加提示](/assets/fluentread/hint.png)

モデルを選択し、「FluentReadにワンクリックで入力」をクリックすると、確認ウィンドウがポップアップ表示され、対応する情報が正しいか確認できます。

![確認](/assets/fluentread/confirm.png)

インポートを確認すると、FluentReadのNew API設定が有効になります。

![配置结果](/assets/fluentread/fluentread.png)

### FluentReadで手動設定

![手动配置](/assets/fluentread/configuration.png)

| 設定項目     | 内容                       |
| ---------- | -------------------------- |
| 翻訳サービス   | NewAPI                     |
| アクセストークン   | NewAPI キー                |
| NewAPIインターフェース | NewAPIデプロイアドレス（/v1なし）  |
| モデル       | リストから選択、またはカスタムモデル |
| カスタムモデル | モデル名                   |
