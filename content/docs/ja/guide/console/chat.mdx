## Lobe Chat

<Callout type="info">
  Lobe Chat は、オープンソースのマルチモーダル対話アプリケーションで、様々な大規模言語モデル（LLM）とプラグイン拡張をサポートしています。テキスト対話だけでなく、画像や音声などのマルチモーダルなインタラクションもサポートしており、パーソナルアシスタント、知識Q&A、コンテンツ作成など、様々なシナリオに適しています。Lobe Chat は、シンプルで直感的なインターフェースを持ち、マルチデバイス同期をサポートしており、ユーザーはニーズに応じてモデルやプラグインをカスタマイズし、対話能力を柔軟に拡張できます。開発者でも一般ユーザーでも、簡単に使いこなせ、スマートな対話による効率的な体験を享受できます。
</Callout>

<Callout type="warn">
  Lobe Chat は現在、New API からのワンクリック設定インポートをサポートしていません。手動で入力する必要があります。
</Callout>

## AI as Workspace

<Callout type="info">
  AI as Workspace は、AI機能をワークスペースに統合する革新的な方法です。AIアシスタントを日常のオフィス業務、コラボレーション、知識管理などのシナリオに深く組み合わせることで、ユーザーはより効率的に情報を処理し、反復的なタスクを自動化し、インテリジェントな提案を得て、全体的な作業効率とエクスペリエンスを向上させることができます。
</Callout>

<Callout type="info">New API からのワンクリック設定インポートをサポートしています</Callout>

## AMA 问天

<Callout type="info">
  AMA 问天 は New API が提供するスマートQ&Aアシスタントで、多段階の対話と複雑な質問への回答をサポートしています。ユーザーは自然言語で AMA 问天 と交流し、知識、技術サポート、またはビジネスコンサルティングを得ることができ、コミュニケーション効率とエクスペリエンスを向上させます。
</Callout>

<Callout type="warn">まず AMA 问天 アプリをインストールする必要があります</Callout>

## OpenCat

<Callout type="info">
  OpenCat は、オープンソースのマルチプラットフォームAIチャットクライアントで、様々な大規模言語モデルへの接続をサポートしています。ユーザーはシンプルなインターフェースを通じてAIと自然言語で対話でき、日常のコミュニケーション、知識Q&A、コンテンツ作成など、様々なシナリオに適しています。OpenCat はマルチデバイス同期をサポートし、柔軟な設定が可能で、個人およびチームでの使用に適しています。
</Callout>

<Callout type="warn">まず OpenCat アプリをインストールする必要があります</Callout>
