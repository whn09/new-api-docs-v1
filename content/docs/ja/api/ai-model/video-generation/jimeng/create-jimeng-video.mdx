即夢公式API形式の動画生成インターフェース。

Actionパラメータを通じて操作タイプを指定できます。
- `CVSync2AsyncSubmitTask`: 動画生成タスクを送信
- `CVSync2AsyncGetResult`: タスク結果を取得

クエリパラメータでActionとVersionを指定する必要があります。
