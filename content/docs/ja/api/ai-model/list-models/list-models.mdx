現在利用可能なモデルのリストを取得します。

リクエストヘッダーに基づいて、返されるフォーマットを自動的に識別します。
- `x-api-key` および `anthropic-version` ヘッダーが含まれる場合、Anthropic フォーマットを返します。
- `x-goog-api-key` ヘッダーまたは `key` クエリパラメーターが含まれる場合、Gemini フォーマットを返します。
- その他の場合、OpenAI フォーマットを返します。
