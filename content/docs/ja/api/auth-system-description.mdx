{/* TODO: Convert to Callout - info: 説明 */}
システムは四段階の認証メカニズムを採用しています：公開、ユーザー、管理者、Root
アクセストークンは、**/api** ルートへのアクセスに使用されるトークンであり、**個人設定 - システムアクセストークン** で生成されます。

## 🔐 認証

- 公開：ログイン不要
- ユーザー：ユーザーアクセストークンが必要です（middleware.UserAuth）
- 管理者：管理者アクセストークンが必要です（middleware.AdminAuth）
- Root：最高権限ユーザーのアクセストークンのみが必要です（middleware.RootAuth）
