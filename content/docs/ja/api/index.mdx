## 概要

New API は、**AIモデルインターフェース**と**管理インターフェース**の2つの主要なカテゴリに分けられた完全なRESTful APIインターフェースを提供します。これらのインターフェースを通じて、AI機能の呼び出しとシステム管理機能を実現できます。

<Callout title="オンラインデバッグ">
  [Apifox 演習場](https://apifox.newapi.ai/) にアクセスしてAPIインターフェースをオンラインでテストおよびデバッグするか、以下のAPIドキュメントを参照してください。
</Callout>
