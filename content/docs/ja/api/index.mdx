## 概要

New API は完全な RESTful API を提供し、**AI モデル API** と **管理 API** の 2 つのカテゴリに分かれています。これらの API を使用して、AI 機能の呼び出しとシステム管理を行うことができます。

<Callout title="オンラインデバッグ">
  [Apifox プレイグラウンド](https://apifox.newapi.ai/) にアクセスして API エンドポイントをオンラインでテストするか、以下の API ドキュメントを参照してください。
</Callout>
