## 1Panel パネルのインストール

{/* TODO: Convert to Callout - tip: 1Panel パネルのインストール */} - 1Panel のインストール、デプロイ、および基本機能の紹介については、[1Panel 公式ドキュメント](https://1panel.cn/docs/) を参照してください。- 1Panel のインストールとデプロイが完了したら、表示されたURLに従ってブラウザを開き、1Panel パネルに入ります。

## New API のインストール

{/* TODO: Convert to Callout - tip: アプリケーションの検索 */}
アプリケーションストアのアプリケーションリストに入り、【AI】カテゴリの下にある New API アプリケーションを見つけてインストールします。

![1panel-newapi](/assets/1panel/1panel-newapi.jpg)

{/* TODO: Convert to Callout - tip: アプリケーションのインストール */}
インストールページで New API アプリケーションのパラメータを設定します。

    - 名称：作成する New API アプリケーションの名前
    - バージョン：New API のバージョンを選択
    - ポート：New API アプリケーションのサービスポート
    - タイムゾーン：サーバーのタイムゾーンを入力（デフォルト Asia/Shanghai）
    - 高度な設定：ポートの外部アクセスに必ずチェックを入れてください

![newapi-setting](/assets/1panel/newapi-setting.jpg)

{/* TODO: Convert to Callout - tip: インストールの確認 */}
確認をクリックしてインストールを開始すると、ページは自動的にインストール済みアプリケーションリストにジャンプし、New API アプリケーションの状態が「起動済み」に変わるのを待ちます。

![newapi-start](/assets/1panel/newapi-start.jpg)

## New API へのアクセス

{/* TODO: Convert to Callout - tip: アプリケーションへのアクセス */} - `インストール済み` ページで、`ジャンプ` をクリックすると New API の WebUI ページに入ることができます。- 使用前に、`パネル設定` ページで `デフォルトアクセスアドレス` を設定しておくことを推奨します。

![newapi-visit](/assets/1panel/newapi-visit.jpg)
