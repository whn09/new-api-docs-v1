## 1Panel パネルのインストール

<Callout type="info" title="1Panel パネルのインストール">
  - 1Panel のインストール、デプロイ、および基本機能の紹介については、[1Panel
  公式ドキュメント](https://1panel.cn/docs/) を参照してください。- 1Panel
  のインストールとデプロイが完了したら、表示されたURLに従ってブラウザを開き、1Panel
  パネルに入ります。
</Callout>

![1panel](/assets/1panel/1panel.jpg)

## New API のインストール

<Callout type="info" title="アプリケーションの検索">
  アプリケーションストアのアプリケーションリストに入り、【AI】カテゴリの下にある
  New API アプリケーションを見つけてインストールします。
</Callout>

![1panel-newapi](/assets/1panel/1panel-newapi.jpg)

<Callout type="info" title="アプリケーションのインストール">
  インストールページで New API アプリケーションのパラメータを設定します。
</Callout>

    - 名称：作成する New API アプリケーションの名前
    - バージョン：New API のバージョンを選択
    - ポート：New API アプリケーションのサービスポート
    - タイムゾーン：サーバーのタイムゾーンを入力（デフォルト Asia/Shanghai）
    - 高度な設定：ポートの外部アクセスに必ずチェックを入れてください

![newapi-setting](/assets/1panel/newapi-setting.jpg)

<Callout type="info" title="インストールの確認">
  確認をクリックしてインストールを開始すると、ページは自動的にインストール済みアプリケーションリストにジャンプし、New
  API アプリケーションの状態が「起動済み」に変わるのを待ちます。
</Callout>

![newapi-start](/assets/1panel/newapi-start.jpg)

## New API へのアクセス

<Callout type="info" title="アプリケーションへのアクセス">
  - `インストール済み` ページで、`ジャンプ` をクリックすると New API の WebUI
  ページに入ることができます。- 使用前に、`パネル設定` ページで
  `デフォルトアクセスアドレス` を設定しておくことを推奨します。
</Callout>

![newapi-visit](/assets/1panel/newapi-visit.jpg)
