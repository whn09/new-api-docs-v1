## 💡 私たちの理念

New API が生まれた理由はシンプルです：**より多くの人々が AI を利用できるようにし、AI へのハードルを下げること**。

AI は一部の人だけの特権ではなく、誰もが手の届くツールになるべきだと私たちは信じています。個人開発者、スタートアップチーム、大企業を問わず、合理的なコストで世界最先端の大規模モデル機能にアクセスできるべきです。

## 🤝 商業化について

私たちが商業化を選んだのは、利益の最大化のためではなく、プロジェクトが**健全で持続可能な発展**を遂げるためです。

サーバーにはコストがかかり、メンテナンスには労力が必要であり、エコシステムの構築にはリソースが必要です。適度な商業化は、New API がコミュニティに継続的にサービスを提供するための必要な保証です。収益は製品の改善、コミュニティの構築、エコシステムの拡大に投資し、すべてのサポートがより良いサービスに変わることをお約束します。

## 📜 紳士協定

New API コミュニティは常に**紳士協定**の理念を守っています：

- すべてのユーザーを信頼し、柔軟でオープンな利用方法を提供します
- 煩雑な制限を設けず、皆様がリソースを合理的に使用することを信じています
- ユーザーが恩恵を受けると同時に、プロジェクトの持続可能な発展を尊重していただくことを期待しています
- New API によってプロジェクトが恩恵を受けた場合は、どのような形でもコミュニティへの還元を歓迎します

これは信頼と尊重に基づく約束であり、オープンソース精神の継承でもあります。
