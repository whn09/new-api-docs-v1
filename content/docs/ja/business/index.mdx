## 💡 私たちの原点

New API が誕生したきっかけはシンプルです。**より多くの人々がAIを利用できるようにし、AIへの参入障壁を下げること**です。

私たちは、AIが一部の特権ではなく、誰もがアクセスできるツールであるべきだと信じています。個人開発者、スタートアップチーム、大企業を問わず、誰もが手頃なコストで世界をリードする大規模モデルの能力にアクセスできるべきです。

## 🤝 商業化について

私たちが商業化を選択したのは、利益の最大化を追求するためではなく、プロジェクトが**健全かつ持続的に発展できる**ようにするためです。

サーバーにはコストがかかり、メンテナンスには労力が必要であり、エコシステム構築にはリソースが必要です。適度な商業化は、New APIがコミュニティにサービスを提供し続けるための不可欠な保証です。私たちは収益を製品の改善、コミュニティの構築、エコシステムの拡大に投資し、皆様からのあらゆるサポートがより良いサービスへと還元されることをお約束します。

## 📜 紳士協定

New API コミュニティは常に**紳士協定**の理念を堅持しています。

- 私たちはすべてのユーザーを信頼し、柔軟でオープンな利用方法を提供します。
- 私たちは煩雑な制限を設けず、皆様がリソースを合理的に利用すると信じています。
- 私たちは、ユーザーが利益を得ると同時に、プロジェクトの持続可能な発展を尊重することを期待しています。
- もしあなたのプロジェクトがNew APIによって利益を得た場合、どのような形でもコミュニティに還元することを歓迎します。

これは信頼と尊重に基づく約束であり、オープンソース精神の継続でもあります。
